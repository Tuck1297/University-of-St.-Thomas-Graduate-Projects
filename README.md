# University of St. Thomas Graduate Projects

This repository serves as a centralized location for graduate-level coursework, research, and technical projects completed at the University of St. Thomas.

## Projects

### 1. GIS-Data-Migration-Project
- **Purpose**: A comprehensive data engineering and visualization project for a database course final.
- **Description**: This project implements a full-stack spatial data pipeline. It features:
    - **Data Ingestion**: Python scripts to download zipped spatial data and ingest it into **DuckDB**.
    - **Migration**: ETL processes to move and transform data from DuckDB to a **PostgreSQL/PostGIS** database.
    - **API Backend**: A **Django** REST API serving spatial data from the PostgreSQL database.
    - **Frontend**: A **React** application with an interactive map to visualize the results.
